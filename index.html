projects - Constancy
mzcloud - main project

mzcloud isn't only a cloud service, it's specialized on dividing the privileges of storing data between the users. It can solve the problem of using a standard cloud service to store public notes and texts in a College. Think about that, what kind of mess is to manage privileges on every single professor?
Technically: it's a php-based web app, no frameworks were used, RSA and 3-DES encryption on API communication, double encryption (md5,Bcrypt) used to store sensible data, compression of pdf type data to ensure the greatest saving of storage memory and such more.
